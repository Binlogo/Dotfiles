# Dotfiles

Build my macOS

## Location Notes

  - `fish` - `~/.config/fish` folder
  - `git` - `~/.gitconfig`
  - `vim` - `~/.vimrc` and the `~/.vim` folder
  - `ssh` - `~/.ssh/config`
  - `tmux` - `~/.tmux.conf`

## Install Homebrew

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```
